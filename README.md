# 💰 Expense & Budget Visualizer
Aplikasi pencatatan keuangan sederhana untuk memantau rencana anggaran (Budget Plan) vs realisasi pengeluaran bulanan.

## ✨ Fitur Utama (MVP)
- **Input Pengeluaran:** Mencatat nama barang, nominal, dan kategori (Food, Transport, Fun).
- **Dashboard Summary:** Melihat total saldo secara otomatis.
- **Visualisasi Data:** Grafik Pie Chart (Chart.js) untuk melihat distribusi pengeluaran.
- **LocalStorage:** Data tetap tersimpan aman di browser meskipun halaman di-refresh.

## 🔍 Fitur Khusus: "Detektif Keuangan"
Selain fitur standar, aplikasi ini dilengkapi dengan:
- **Budget Planning:** Kolom khusus untuk memasukkan target budget tiap kategori.
- **Leak Detection:** Membandingkan rencana budget dengan realisasi untuk melihat "kebocoran" keuangan.
- **Visual Alert:** Warna akan berubah jika pengeluaran melebihi anggaran yang direncanakan.

## 🛠️ Teknologi yang Digunakan
- **HTML5** & **Tailwind CSS** (Responsive Design untuk Laptop & HP)
- **Vanilla JavaScript** (Logic & LocalStorage)
- **Chart.js** (Data Visualization)

## 🚀 Cara Menggunakan
1. Masukkan rencana budget kamu di setiap kategori.
2. Tambahkan pengeluaran harian kamu melalui form.
3. Pantau grafik dan summary di bagian atas untuk memastikan keuanganmu tidak "bocor".
